# AstrBot 鹿管签到插件 (Deer Pipe)

一款精简、美观的签到插件，移植自 Koishi 插件 `deer-pipe`。支持个人签到、助人签到、补签及可视化日历和排行榜。

## 特性

- **美观渲染**：基于 HTML 渲染的签到日历与排行榜，支持多种背景预设。
- **灵活配置**：支持自定义每日签到上限、助人上限，并可选每月自动重置总次数。
- **补签与戒签**：本月内的记录可随时补签（免费）或取消。

## 指令

| 指令            | 别名    | 说明                                         |
| :-------------- | :------ | :------------------------------------------- |
| `/鹿`           | `/🦌`   | 每日签到（支持多次，视配置而定）             |
| `/帮鹿 @用户`   | -       | 帮他人签到，增加对方的总签到次数             |
| `/看鹿 [@用户]` | -       | 查看自己或指定用户的本月签到日历图           |
| `/补鹿 [日期]`  | -       | 补签本月内的指定日期                         |
| `/戒鹿 [日期]`  | -       | 取消指定日期的签到记录（省略日期则取消今日） |
| `/鹿榜 [年\|总]` | `/🦌榜` | 查看排行榜。无参数为月榜，支持年榜、总榜     |

## 配置

可以在 AstrBot 管理面板中配置以下项：

- **每日签到次数上限**：默认为 3 次。设置为 `-1` 则不限制。
- **每日帮他人签到次数上限**：默认为 5 次。设置为 `-1` 则不限制。
- **日历背景预设**：可选 `鹿管` 或 `心奈` 视觉风格。

## 存储

- **数据库**：`data/plugin_data/astrbot_plugin_deer_pipe/deer_pipe.db` (SQLite)
- **配置**：自动载入管理面板设置。
